# 图片资源目录

此目录用于存放网站所需的图片资源。

## 需要的图片

### 头像图片
- **文件名**: `avatar-placeholder.jpg`
- **尺寸**: 建议 400x400 像素或更高（正方形）
- **格式**: JPG 或 PNG
- **用途**: 在"关于我"页面显示个人头像

如果未上传头像图片，网站会自动显示一个带有"DC"字母的圆形占位符。

## 如何添加头像

1. 准备一张正方形头像图片（建议 400x400 像素或更高）
2. 将图片命名为 `avatar-placeholder.jpg`
3. 放入此目录（`assets/images/`）
4. 刷新浏览器即可看到效果

## 其他图片

你也可以在此目录添加：
- 项目截图
- 技能图标
- 其他需要的图片资源

记得在 HTML 文件中更新相应的图片路径。
